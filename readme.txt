Projekt "MasterData" umożliwia analizę danych. Opcja analizy danych wczytanych z pliku zewnętrznego jest jeszcze niedostępna.
Można jedynie używać drugiej opcji, czyli stworzenia tabeli z danymi od zera i ich analizy. 

Dostępne są 3 opcje analizy:
1. Analiza wybranej kolumny. Szczegółowe opcje analizy zależą od typu danych w wybranej kolumnie.
2. Wyszukanie wierszy, które spełniają określone warunki. 
3. Stworzenie wykresu. Dostępne są 4 typy wykresów. Każdy z nich ma menu z podstawowymi opcjami np. określenie tytułu wykresu, wybór koloru itp.

Użyte technologie: Python, CSS i BOOTSTRAP.

Aplikacja ma budowę modułową. Głównym plikiem jest app.py. Folder views zawiera pliki .py dedykowane poszczególnym funkcjonalnościom aplikacji. 

Folder assets zawiera pliki .css ze stylami poszczególnych sekcji projektu. Zawiera również użyte w projekcie grafiki.

W projekcie niezbędne było użycie wyrażeń regularnych.

Projekt jest responsywny.
